Nome do Projeto: Verdear: Sua vitrine para o campo

DESCRIÇÃO DO PROJETO O projeto Verdear consiste no desenvolvimento de uma plataforma de e-commerce que conecta pequenos agricultores e consumidores, permitindo a comercialização de produtos orgânicos de forma simples e direta. Permitindo que os produtores cadastrem, gerenciem e vendam seus produtos de forma simples e acessível. A aplicação contará com módulos distintos para produtores e clientes, abrangendo funcionalidades de cadastro, gestão de pedidos, compras e avaliações.

Integrantes: Ana Girelli Bruno Teles João Rickli Julia Goulart Tais Carvalho
